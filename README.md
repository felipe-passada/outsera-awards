# OutSera - Análise de Intervalos de Prêmios

Aplicação Spring Boot que calcula e analisa os intervalos entre prêmios de produtores de filmes com base em dados da Golden Raspberry Awards.

## Como Rodar o Projeto

### 1. Clonar o Repositório

```bash
git clone <url-do-repositorio>
cd outsera
```

### 2. Compilar e Executar

```bash
mvn clean install
mvn spring-boot:run
```

### 3. Testes de Integração

```bash
mvn test
```
